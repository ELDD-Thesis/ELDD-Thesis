Firmware Repository

This firmware powers a real-time ECU data display using an ESP32-S3. It supports K-Line (DS2) for older BMWs and CAN-Bus for newer ones. Sensor data is processed and visualized on an LCD.

BimmerDis

A disassembler for BMW’s proprietary "BEST" language used in EDIABAS. Enables low-level analysis of ECU diagnostic logic.

BimmerGram

Tool for converting BMW .b1v parameter files into structured C code for integration with diagnostic software.

BimmerJson

Transforms decompiled BimmerDis outputs into clean, grouped JSON for human-readable analysis.
